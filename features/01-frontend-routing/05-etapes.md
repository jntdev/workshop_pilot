# 05 - Étapes de réalisation (frontend)

1. Mettre en place `MainLayout` (et `ChapterLayout` si retenu).
2. Déclarer les routes `/`, `/clients`, `/atelier`, `/location`.
3. Créer le composant dashboard (cards/boutons vers les chapitres, pas de CTA additionnel).
4. Structurer le dossier SCSS comme décrit, écrire variables + mixins (sans skeletons).
5. Écrire les styles de chaque page (dashboard + chapitres) en appliquant les règles couleurs.
6. Ajouter tests frontend (ex. React Testing Library / Vue Test Utils) pour vérifier routing et rendu du dashboard.
