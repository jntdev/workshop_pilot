# 01 - Périmètre métier

Feature : **Interface CRUD complète pour les clients**.

## Objectif
Permettre la gestion complète des clients via une interface utilisateur intuitive :
- Afficher la liste de tous les clients
- Rechercher un client (front-end)
- Créer un nouveau client
- Modifier les informations d'un client existant
- Supprimer un client

## Pages concernées
- `/clients` : Page liste avec recherche et création
- `/clients/{id}` : Page de détail/modification d'un client

## Interactions
- Navigation fluide entre liste et détail
- Recherche instantanée côté front-end
- Feedback visuel au survol des cartes
- Actions de modification et suppression depuis la fiche client
