# 02 - Statut simplifie

## Etats
- Devis : editable, toutes les informations visibles.
- Facture : lecture seule (non modifiable).

## Transition
- Devis -> Facture (via bouton "Transformer en facture" + confirmation).

## Regles
- Pas de retour Facture -> Devis.
- Suppression des statuts brouillon / pret / modifiable.
