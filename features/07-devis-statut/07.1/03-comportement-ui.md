# 03 - Comportement UI

1. **Formulaire devis**
   - Aucun select de statut ni badge.
   - Toutes les informations sont visibles et modifiables.

2. **Bouton "Transformer en facture"**
   - Affiche uniquement sur un devis.
   - Clic -> pop up de confirmation.
   - Confirmation -> creation de la facture et passage en lecture seule.

3. **Formulaire facture**
   - Tous les champs en lecture seule.
   - Le bouton "Transformer en facture" disparait.

4. **Liste des devis**
   - Actions disponibles : "Consulter" et "Supprimer".
   - La colonne de statut est retiree (option : afficher "Devis" / "Facture").

5. **Suppression**
   - Pop up de confirmation avant suppression.
   - pas de bouton suppression si il s'agit d'une facture.
