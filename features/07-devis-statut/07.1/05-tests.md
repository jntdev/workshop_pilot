# 05 - Tests & validations

1. **Conversion devis -> facture**
   - Le bouton ouvre une confirmation.
   - Annuler ne convertit pas.
   - Confirmer cree une facture et verrouille le formulaire.

2. **Lecture seule facture**
   - Aucun champ editable.
   - Aucune modification sauvegardee.

3. **Devis editable**
   - Toutes les informations visibles et modifiables avant conversion.

4. **Liste des devis**
   - "Consulter" ouvre le devis en edition.
   - "Supprimer" demande confirmation.

5. **Suppression**
   - Annuler la suppression ne supprime rien.
   - Confirmer supprime le devis (ou la facture si autorise).
