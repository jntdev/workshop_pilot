# 02 - Statuts / etats

## Documents
- Devis (PDF)
- Facture (PDF)

## Etats de generation
- Non genere : aucun PDF disponible.
- Genere : PDF disponible, regenere a la demande.

## Regles
- La generation est declenchee par l'utilisateur.
- Le PDF reflete les donnees courantes du devis/facture.
- Pour un devis, le nom du fichier PDF reste identique entre telechargements.
