# 02 - Objectifs métier & produit

1. **Traçabilité de la charge**  
   Chaque ligne de devis doit pouvoir recevoir un temps estimé (visible uniquement en back-office).

2. **Suivi temps réel**  
   Après intervention, l'atelier renseigne le temps réellement passé sur le devis/facture pour piloter les marges horaires.

3. **Édition après facturation**  
   Le champ "temps réel" reste éditable même sur une facture, afin de saisir l'information a posteriori.

4. **Ergonomie**  
   Saisie simple (heures décimales) et calcul automatique du total estimé pour éviter les erreurs/manips Excel.

5. **Base décisionnelle**  
   Les API et exports doivent inclure ces champs pour alimenter rapidement des rapports (marges / heure, écart estimé vs réalisé).
