# Feature 12 — Formulaire de réservation location (panneau droit)

## Rôle dans l'écran Location
- Le formulaire occupe la colonne droite (25 %) de l'écran Location et reste visible quand on parcourt les données.
- Pour cette itération, il fonctionne de manière complètement autonome : aucune interaction ou synchronisation n'est prévue avec le tableau à gauche.
- L'objectif est d'offrir un \"bête\" formulaire complet qui permet de consigner toutes les informations nécessaires dès la prise d'appel.

## Objectifs métier
- Permettre la création rapide d'une réservation, même si le client n'existe pas encore dans la base.
- Conserver l'historique de contact (dates de prise de contact, période de location, logistique livraison/récupération) pour faciliter le suivi.
- Préparer la future génération de documents (devis/facture) et la coordination atelier grâce à des informations structurées (vélo, accessoires, acomptes, statut).
