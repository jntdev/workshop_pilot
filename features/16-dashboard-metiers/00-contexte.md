# Contexte — Dashboard métiers unifié

- Le dashboard actuel affiche trois cartes-lien (Vente, Atelier, Location) qui redirigent vers les autres pages sans fournir d’information utile.  
- L’équipe veut une vue synthétique de la performance de chaque volet (“combien rapporte l’activité ?”) pour piloter rapidement l’entreprise.  
- Feature 16 remplace les cartes par trois colonnes KPI (Vente / Atelier / Location) avec les mêmes indicateurs : chiffre d’affaires, marge brute, panier moyen. Le layout doit être lisible, responsive et prêt à accueillir de futures métriques.
