# Raison métier

1. **Pilotage instantané** : disposer d’un résumé des trois métiers dès l’ouverture de l’outil pour suivre leur contribution respective.  
2. **Alignement équipe / direction** : tout le monde parle des mêmes KPIs (CA, marge, panier) et peut comparer les volets sans ouvrir plusieurs écrans.  
3. **Préparation commerciale** : savoir en un coup d’œil si la Location ou la Vente tourne bien pour prioriser les actions marketing.  
4. **Base pour futures alertes** : ce dashboard central permettra demain d’ajouter des alertes (par exemple “CA Location en baisse vs N-1”).
