# Raison métier

1. **Ne rien perdre** : Chaque note client est enregistrée avec les infos de contact, impossible d'oublier un rappel.

2. **Communication interne** : Le comptoir peut transmettre une info à l'atelier (et vice-versa) sans post-it ni interruption.

3. **Traçabilité** : On sait qui a écrit quoi, quand, et si c'est traité.

4. **Suivi des actions** : Statuts "non lu" → "lu" → "résolu" pour ne rien laisser traîner.

5. **Temps réel** : Notification immédiate quand un message arrive, confirmation de lecture visible par l'expéditeur.

6. **Mode comptoir/atelier** : Chaque poste voit ses messages pertinents, peut basculer pour voir l'autre vue.
